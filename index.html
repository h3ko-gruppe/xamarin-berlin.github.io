Xamarin.Berlin ist ein Gemeinschaft unabhängiger Experten in und um Berlin. Details

Xamarin ist das optimale Tool für Ihre mobilen Unternehmenslösungen

Erfahren Sie mehr über Xamarin von einem unserer Experten. Oder besuchen sie doch einfach einen unserer Workshops.

Cross Platform

Erstellen Sie Apps für iOS, Android und Windows mit nur einer C# Code Basis und verwenden sie über 96% Ihres Codes auf allen Platformen.

100% Nativ

Im Gegensatz zu Cordova und PhoneGap erstellen Sie mit Xamarin 100% native Apps. Ohne Performanceverlust und native UI auf allen Platformen.
